# ZHL-max.github.io

GitHub Pages user site for `https://ZHL-max.github.io/`.

This is a no-build static personal blog homepage for an AI major. It keeps the structure simple:

- `index.html`
- `styles.css`
- `script.js`
- `assets/site-mark.svg`
- `.nojekyll`

Local preview:

```bash
python3 -m http.server 8000
```

Then open `http://127.0.0.1:8000/`.
